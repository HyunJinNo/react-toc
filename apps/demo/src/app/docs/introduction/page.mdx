# Introduction

`react-toc` is a library that automatically generates a Table of Contents (TOC) from your headings with scroll tracking.

## Features

- 🔍 Automatically extracts h2 ~ h6 headings
- 🎯 Active heading tracking (IntersectionObserver)
- 🌳 Nested TOC structure
- 🎨 Fully customizable styles
- ⚡ Smooth scrolling
- 🧩 Works with React / Next.js
