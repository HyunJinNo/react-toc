## Install

Run the following commands in the `React` project directory:

npm:

```bash copy
npm install @hyunjinno/react-toc
```

yarn:

```bash copy
yarn add @hyunjinno/react-toc
```

pnpm:

```bash copy
pnpm add @hyunjinno/react-toc
```

## Quick Start

<Steps>

### Step 1.

Import the CSS file at the entry point of the application

```tsx copy
import "@hyunjinno/react-toc/style.css";
```

### Step 2.

Introduce `TocProvider` and `Toc` components where they need to be used, basic usages:

```tsx copy
"use client";

import { H2, H3, H4, Toc, TocProvider } from "@hyunjinno/react-toc";

export const TocBasic = () => {
  return (
    <TocProvider>
      <div className="flex flex-row justify-between border p-4">
        <div>
          <H2>1. Heading</H2>
          <H2>2. Heading</H2>
          <H3>2.1. Heading</H3>
          <H3>2.2. Heading</H3>
          <H3>2.3. Heading</H3>
          <H4>2.3.1. Heading</H4>
          <H4>2.3.2. Heading</H4>
          <H4>2.3.3. Heading</H4>
          <H3>2.4. Heading</H3>
          <H2>3. Heading</H2>
        </div>
        <Toc />
      </div>
    </TocProvider>
  );
};
```

</Steps>
